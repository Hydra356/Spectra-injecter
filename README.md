# 🚀 Features

- 📂 **DLL Path**
  - Browse and select the DLL to inject.

- 🎯 **Target Process**
  - Display all running processes.
  - Select the target process.
  - Refresh process list instantly.

- ⚙️ **Injection Methods**
  - ✅ Manual Map
  - ✅ Erase PE Header
  - ✅ Stealth injection
- 🚀 **Injection**
  - One-click DLL injection.

- 📜 **Console**
  - Real-time status messages.
  - Success & error logs.
  - Injection progress.

- 🎨 **Modern Interface**
  - Minimalistic dark UI.
  - Fast and responsive.
  - Easy to use.
